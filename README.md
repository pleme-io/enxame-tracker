# enxame-tracker

[![crates.io](https://img.shields.io/crates/v/enxame-tracker.svg)](https://crates.io/crates/enxame-tracker)
[![license](https://img.shields.io/badge/license-MIT-blue.svg)](./LICENSE)

enxame-tracker — a fast Rust BitTorrent tracker server (HTTP announce + scrape, compact peers, in-memory swarm registry). The server half of the pleme-io ENXAME suite; a caixa Servico for HELM-NATIVE deploy.

Part of **ENXAME** — the pleme-io BitTorrent suite: one sans-io engine and
thin faces, with GUIs on every platform (terminal, web, native desktop).
See [`theory/ENXAME.md`](https://github.com/pleme-io/theory/blob/main/ENXAME.md).

## Use

```toml
enxame-tracker = "0.1"
```
